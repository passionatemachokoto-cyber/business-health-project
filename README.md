 Business Health Monitoring System

This project monitors business revenue health using forecasting, alerts, and dashboards.

What it does
- Forecasts revenue
- Detects abnormal drops or spikes
- Raises alerts
- Displays insights in a dashboard

 Tech used
- Python
- Pandas
- Prophet
- FastAPI
- Streamlit

How to run
1. Activate virtual environment
2. Run forecasting script
3. Start API
4. Open dashboard
